Unfortunately, he couldn't solve this problem. Please help him!</p><p>$$$^{\dagger}\text{mex}(c_1, c_2, \ldots, c_k)$$$ is defined as the smallest <span class="tex-font-style-bf">non-negative</span> integer $$$x$$$ which does not occur in the sequence $$$c$$$.
