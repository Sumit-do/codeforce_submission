## Problem Statement

<div class="header"><div class="title">B. Recollect Numbers</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>There are $$$2n$$$ cards with numbers $$$1,1,2,2,\ldots,n,n$$$ written on them. In other words, for all $$$j=1,2,\ldots,n$$$, there are <span class="tex-font-style-bf">exactly</span> $$$2$$$ cards with the number $$$j$$$. Each card only has one number written on its front.</p><p>You will play a card flipping game. Initially, all $$$2n$$$ cards are placed on their back (the side without numbers). In each turn, you flip <span class="tex-font-style-bf">exactly two</span> cards. If the two cards have the same number, you <span class="tex-font-style-bf">discard</span> the two cards. Otherwise, you flip them back to their original position. You win when all $$$2n$$$ cards are discarded. Note that you do <span class="tex-font-style-bf">not</span> have to flip the two cards simultaneously, so you can decide on your choice of the second card after seeing the number on the first one.</p><p>Consider the following "greedy" algorithm to play the game. Initially, the $$$2n$$$ cards are placed in a row arbitrarily. Then your strategy on each turn is as follows:</p><ul> <li> If there are two cards that <span class="tex-font-style-bf">you have flipped previously</span> and <span class="tex-font-style-bf">have the same number</span>, flip those two cards. </li><li> Otherwise, flip the <span class="tex-font-style-bf">first card</span>$$$^\text{*}$$$ that you have never flipped so far as the first one. Let's say this card has the number $$$x$$$. <ul> <li> Afterwards, if there is another card that you have flipped previously and <span class="tex-font-style-bf">has the number $$$x$$$</span>, flip that card. </li><li> Otherwise, flip the <span class="tex-font-style-bf">first card</span>$$$^{\text{∗}}$$$ that you have never flipped so far (<span class="tex-font-style-bf">including in this turn</span>) as the second one. </li></ul> </li></ul><p>It can be shown that the algorithm's strategy is uniquely determined on every turn.</p><p>You must solve the following problem regarding the algorithm stated above.</p><ul> <li> Given $$$n$$$ and $$$k$$$, please find an orientation of the $$$2n$$$ cards for which the algorithm above takes <span class="tex-font-style-bf">exactly</span> $$$k$$$ turns to win the game. </li></ul><p>Additionally, if such an orientation does not exist, please report so.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$Here, "the first card" of some condition refers to the earliest card in the row satisfying that condition.</p></div></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^3$$$). The description of the test cases follows. </p><p>The only line of each test case contains two integers $$$n$$$ and $$$k$$$ ($$$1 \le n \le 300\,000$$$, $$$1 \le k \le 1\,000\,000$$$).</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$300\,000$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>If there is an orientation of cards that satisfies the conditions, output "<span class="tex-font-style-tt">YES</span>" on a new line.</p><p>Then, output $$$2n$$$ integers $$$a_1,a_2,\ldots,a_{2n-1},a_{2n}$$$ on the next line. Here, $$$a_i$$$ is the number written on the $$$i$$$-th card.</p><p>Do note that the sequence $$$a$$$ must satisfy the following conditions:</p><ul> <li> For each $$$1 \le i \le 2n$$$, $$$1 \le a_i \le n$$$; </li><li> For each $$$1 \le j \le n$$$, $$$j$$$ appears in $$$a$$$ <span class="tex-font-style-bf">exactly</span> twice; </li><li> When the cards are placed in this order, the algorithm stated above takes exactly $$$k$$$ turns to win the game. </li></ul><p>If there are multiple solutions, print any of them.</p><p>If there is no orientation of cards that satisfies the conditions, output "<span class="tex-font-style-tt">NO</span>" on a separate line.</p><p>You can output the answer in any case.
